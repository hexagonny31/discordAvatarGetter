### Things you need!
1. Your **OWN** discord bot token.
2. A Discord user ID.

## How to get your very own discord bot token
1. Go to [Discord Developer Portal](https://discord.com/developers/applications) website.
2. Create an Application by clicking **New Application**.
3. Go to **Bot** tab under Overview.
4. Find **Token** section and click **Reset Token**.
5. After successfully authenticating, click the **Copy** button.

After copying, you can now paste it inside the script. Make sure to have quotation marks around it.<br>

## How to get someone's user ID without asking
1. Go to Discord.
2. Open **User Settings** menu.
3. Find **Developer Section** and toggle **Developer Mode**.
4. Find a user.
5. View their whole profile.
6. Click the **three dotted** button.
7. Click **Copy User ID** button.<br>
